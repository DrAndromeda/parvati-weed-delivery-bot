# 🌿 Parvati Weed Thailand Bot 🚀

Premium cannabis delivery Telegram bot for Thailand.

## Features
- 🛍️ **Premium catalog** — Spar City strains with grades (4A/5A+)
- 🏆 **Grade badges** & strain types (Sativa/Indica/Hybrid)
- 💰 **Multi-size pricing** — 1g / 3.5g / 7g / 14g / 28g
- 🛒 **Smart cart** — add/remove/clear, quantity controls
- 🌍 **Bilingual** — English / Русский
- 💳 **Payment**: PromptPay QR · Cash to courier · Crypto (USDT/BTC)
- 📍 **Delivery zones**: Bangkok, Phuket, Samui, Phangan, Krabi & more
- 👤 **Admin notifications** — full order details in DM

## Quick Start

```bash
BOT_TOKEN=your_token ADMIN_ID=237228075 npm start
```

## Deploy on VPS

```bash
npm install
nohup env BOT_TOKEN=xxx ADMIN_ID=237228075 npm start > bot.log 2>&1 &
```
